# API Security Risk Analysis

## 📌 Overview
This project presents a security analysis of a public API using Postman.

## 🛠 Tools Used
- Postman
- Browser Developer Tools

## 🔍 API Tested
- JSONPlaceholder
- https://jsonplaceholder.typicode.com

## 📊 Methodology
- Tested API endpoints
- Analyzed responses and headers
- Identified security risks
- Suggested remediation

## ⚠️ Key Findings
- No Authentication
- Excessive Data Exposure
- Broken Authorization
- No Rate Limiting
- Missing Security Headers

## 📁 Files Included
- API Security Risk Analysis Report (PDF)
- Screenshots of API testing

## 🎯 Conclusion
The analysis highlights the importance of securing APIs to prevent data exposure and unauthorized access.
